Broken snippet (preview)

<html>
<head>
  <title> John Cena Fan Page
  <meta charset="utf-8">
</head>
<body>
  <head>
    <meta name="viewport" content="width=device-width, initial-scale=1">

  <footer>

</body>
</html>

Checklist: find these issues

  • Invalid DOCTYPE
  • Unclosed <title> / misplaced tags in <head>
  • Extra <head> inside <body>
  • Duplicate id values
  • Mis-nested & unclosed tags (<p>, <strong>)
  • Image missing alt, unquoted width
  • Broken link protocol + wrong target
  • List items not closed
  • Stylesheet link inside body
  • Footer not closed

Step 1: Fix the DOCTYPE

Before

<DOCTYPE html>

After

<!DOCTYPE html>

Ensures standards mode rendering.

Step 2: Clean up the <head>

Before

<head>
  <title> John Cena Fan Page
  <meta charset="utf-8">
</head>
...
<body>
  <head> <meta name="viewport" ... > </head>

After

<head>
  <meta charset="utf-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1" />
  <title>John Cena Fan Page</title>
  <link rel="stylesheet" href="styles.css" />
</head>

All metadata + styles sit in the <head>.

Step 3: Headings & IDs

Before

<h1 id="title">Welcome...</h1>
<h2 id="title">About John</h2>

After

<h1 id="title">Welcome...</h1>
<h2>About John</h2>

Use a unique id or none for secondary headings.

Step 4: Fix nesting & paragraphs

Before

<p><strong>Hi, I am John Cena.</p></strong>

After

<p><strong>Hi, I am John Cena.</strong></p>

Tags must close in the reverse order they open.

Step 5: Image & link attributes

Before

<img src="cena.jpg" width=300>
<a href="htp://wwe.com" target="blank">Official WWE Page</a>

After

<img src="https://upload.wikimedia.org/wikipedia/commons/5/59/John_Cena_2012.jpg" alt="Photo of John Cena" width="300" />
<a href="https://www.wwe.com/superstars/john-cena" target="_blank" rel="noopener noreferrer">Official WWE Page</a>

Add alt, quote numeric attributes, use https, and safe target.

Step 6: Lists & closing tags

Before

<ul>
  <li>Champion<li>Actor</li>
  <li>Philanthropist
</ul>
...
<footer>
  <p>© 2025 John Cena

After

<ul>
  <li>Champion</li>
  <li>Actor</li>
  <li>Philanthropist</li>
</ul>
...
<footer>
  <p>© 2025 John Cena Fan Page</p>
</footer>

Every opened tag should be properly closed.

Final fixed HTML

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="utf-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1" />
  <title>John Cena Fan Page</title>
  <link rel="stylesheet" href="styles.css" />
</head>
<body>
  <h1 id="title">Welcome to John Cena’s Page</h1>
  <h2>About John</h2>

  <p><strong>Hi, I am John Cena.</strong></p>
  <p>I like wrestling, movies, and memes.</p>

    <a href="https://www.wwe.com/superstars/john-cena" target="_blank" rel="noopener noreferrer">
      Official WWE Page

</html>

- Close and properly nest all tags
- Ensure unique ids; add alt text
- Move CSS <link> to <head>
- Use https:// and target="_blank" + rel="noopener noreferrer"
