What is a Web API?

An API is like a waiter üßë‚Äçüç≥ between your frontend and database ‚Äî it takes requests and returns responses.

Frontend  <---->  Express.js API  <---->  Database

It allows different applications to communicate using HTTP methods like GET, POST, PUT, PATCH, DELETE.

HTTP Methods Overview üåê

  • GET ‚Äì Retrieve data
  • POST ‚Äì Add new data
  • PUT ‚Äì Replace existing data
  • PATCH ‚Äì Update part of data
  • DELETE ‚Äì Remove data
GET /products        ‚Üí Fetch all products
POST /products       ‚Üí Add a product
PUT /products/:id    ‚Üí Replace a product
PATCH /products/:id  ‚Üí Update partial data
DELETE /products/:id ‚Üí Remove a product

Setting Up Express.js üß©

# Terminal commands
mkdir week5-api
cd week5-api
npm init -y
npm i express
// index.js
import express from "express";

const app = express();
app.use(express.json()); // Middleware to parse JSON

app.listen(4000, () => console.log("Server running on http://localhost:4000"));

Creating Routes

app.get("/api/products", (req, res) => {
  res.json(["Laptop", "Phone", "Tablet"]);
});

app.post("/api/products", (req, res) => {
  const newProduct = req.body;
  res.json({ message: "Product added", product: newProduct });
});

app.put("/api/products/:id", (req, res) => {
  res.json({ message: `Replaced product ${req.params.id}` });
});

app.patch("/api/products/:id", (req, res) => {
  res.json({ message: `Updated product ${req.params.id}` });
});

app.delete("/api/products/:id", (req, res) => {
  res.json({ message: `Deleted product ${req.params.id}` });
});

Parameters & Query Strings üîç

// URL parameter example
app.get("/api/users/:userId", (req, res) => {
  const userId = req.params.userId;
  res.json({ message: `User ID received: ${userId}` });
});

// Query string example
app.get("/api/search", (req, res) => {
  const { q } = req.query;
  res.json({ results: `You searched for '${q}'` });
});

Try / Catch for Error Handling üß†

Use try/catch to safely handle errors and prevent server crashes.

app.get("/api/error-demo", (req, res) => {
  try {
    let result = riskyFunction(); // This will throw an error
    res.json({ result });
  } catch (error) {
    res.status(500).json({ error: "Something went wrong!" });
  }
});
