Declaring Variables

We use var, let, and const to store data in JavaScript.

var x = 10;     // function-scoped, can be re-declared
let y = 20;     // block-scoped, cannot be re-declared in same scope
const z = 30;   // block-scoped, constant value

x = 15; // ‚úÖ allowed
z = 50; // ‚ùå error - const is read-only

Data Types

JavaScript is dynamically typed. Common data types include:

  • String ‚Üí "Hello World"
  • Number ‚Üí 42
  • Boolean ‚Üí true / false
  • Null ‚Üí intentional empty value
  • Undefined ‚Üí variable declared but not assigned
  • Object ‚Üí key-value pairs
  • Array ‚Üí list of values
let product = "Laptop";
let price = 120000;
let available = true;
let discount = null;
let features = ["Touchscreen", "Backlit Keyboard"];
let manufacturer; // undefined

console.log(typeof product, typeof discount); // string, object

Note: typeof null returns 'object' ‚Äì a long-standing JavaScript quirk.

Loops: for & forEach üîÅ

Loops help repeat actions for multiple items in an array or range.

// for loop
for (let i = 0; i < 5; i++) {
  console.log("Count:", i);
}

// forEach loop
const fruits = ["Apple", "Banana", "Mango"];
fruits.forEach(fruit => {
  console.log(fruit);
});

Array Manipulation üç±

let numbers = [1, 2, 3, 4, 5];

numbers.push(6);       // Add at end
numbers.pop();         // Remove from end
numbers.unshift(0);    // Add at start
numbers.shift();       // Remove from start

let doubled = numbers.map(num => num * 2);   // [2, 4, 6, 8, 10]
let evens = numbers.filter(num => num % 2 === 0); // [2, 4]

console.log(numbers);
console.log(doubled);
console.log(evens);

Introduction to Node.js üß©

JavaScript originally ran only in browsers. Node.js changed that by allowing JS to run on servers, desktops, and the cloud.

// hello.js
console.log("Running JavaScript outside the browser!");
# Run in terminal
node hello.js

Node.js is built on Chrome's V8 engine and is widely used to build APIs, web servers, automation tools, and real-time applications.
