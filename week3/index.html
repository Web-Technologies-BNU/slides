What is CSS?

Cascading Style Sheets ‚Äî the presentation layer of the web.

  • Controls layout, color, spacing, typography, and responsiveness.
  • ‚ÄúCascading‚Äù means rules can inherit and override based on specificity and order.
  • Separation of concerns: HTML = structure, CSS = style, JS = behavior.
/* selector */
.card {
  background: #111827;
  color: #e5e7eb;
  padding: 1rem;
}

Syntax: selector ‚Üí { property: value; }

How do you add CSS to a page?

Three common ways ‚Äî pick the right tool for the job.

  1. External file ‚Äî best for real projects 
    <link rel="stylesheet" href="styles.css">
  2. Internal <style> ‚Äî fine for small demos 
    <style>
  body { font-family: system-ui; }
</style>
  3. Inline ‚Äî quick one‚Äëoffs 
    <h1 style="color:tomato">Hi</h1>
Avoid
  • Overusing @import ‚Äî it delays loading.
  • Massive inline styles ‚Äî hard to maintain.
  • Unscoped rules that accidentally style everything.

Centering a <div> ‚Äî 3 useful patterns

Different context, different tool.

/* 1) Horizontal in a fixed-width container */
.container { width: 600px; margin: 0 auto; }

/* 2) Full center with Flexbox */
.stage { display:flex; align-items:center; justify-content:center; }

/* 3) Absolute + transform */
.box { position:absolute; top:50%; left:50%;
       transform: translate(-50%, -50%); }

Flexbox is usually the simplest for full centering.

Fonts: stacks & fallbacks

Always give the browser a safe plan B.

/* System UI stack: fast & good */
body{
  font-family: system-ui, -apple-system, Segoe UI, Roboto,
               Ubuntu, Cantarell, Noto Sans, Helvetica, Arial,
               "Apple Color Emoji", "Segoe UI Emoji";
}

/* With a Google Font loaded via <link> */
body{ font-family: "Inter", system-ui, Arial, sans-serif; }
Load a web font
<!-- In <head> -->
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;700&display=swap" rel="stylesheet">

Use display=swap to avoid invisible text during load.

Why CSS this way?

Practical benefits you‚Äôll feel as projects grow.

  • Maintainability: one stylesheet powers many pages.
  • Performance: external CSS is cached across pages.
  • Consistency: tokens & variables create a design system.
  • Accessibility: proper sizing/contrast improves readability.
  • Teamwork: clean separation from HTML & JS.
Debug tips
  • Use DevTools: toggle CSS rules live.
  • Check computed styles to see what wins specificity.
  • Add temporary outlines: *{outline:1px solid #fff3}
