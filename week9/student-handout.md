## ðŸŽ¨ The Six Core UX Principles

### 1. Usefulness
**Does it solve a real problem?**
- Ask: "What value does this provide to users?"
- Example: Restaurant site prioritizes menu, hours, location

### 2. Usability
**Can users accomplish their goals easily?**
- Test: Can a new user complete key tasks without help?
- Example: Clear "Add to Cart" buttons, simple checkout

### 3. Findability
**Can users locate what they need?**
- Rule: Important content reachable in â‰¤ 3 clicks
- Example: Logical navigation, search functionality

### 4. Credibility
**Does it look trustworthy?**
- Include: Security badges, reviews, clear contact info
- Example: Professional design, HTTPS, verified testimonials

### 5. Desirability
**Is it aesthetically pleasing?**
- Consider: Color scheme, typography, imagery
- Example: Consistent branding, quality visuals

### 6. Accessibility
**Can everyone use it?**
- Standard: WCAG 2.1 compliance
- Example: Alt text, keyboard navigation, color contrast

## ðŸ’¡ Powerful AI Prompts

### Keyword Research
```
"Generate 20 long-tail keyword ideas for [TOPIC] that show buyer intent. 
For each keyword, indicate:
- Estimated search volume (high/medium/low)
- Search intent (informational/transactional/navigational)
- Difficulty level (easy/moderate/hard)"
```

### Blog Outline
```
"Create a comprehensive blog post outline for '[KEYWORD TOPIC]' targeting 
the keyword '[TARGET KEYWORD]'. Include:
- 3 compelling title options
- Introduction hook
- 6-8 main sections with H2 headings
- Key points for each section
- Conclusion with call-to-action
- Suggested internal linking opportunities"
```

### Meta Descriptions
```
"Write 5 variations of meta descriptions for a page about [TOPIC]. 
Each should:
- Be 150-160 characters
- Include the keyword '[TARGET KEYWORD]'
- Use action words to encourage clicks
- Clearly state the page value"
```

### Ad Copy
```
"Create 5 versions of Google ad copy for [PRODUCT/SERVICE]. Requirements:
- Headline: under 30 characters, include '[KEYWORD]'
- Description: under 90 characters
- Emphasize: [UNIQUE SELLING POINT]
- Include clear call-to-action"
```

### Usability Analysis
```
"Analyze this website screenshot for usability issues. Evaluate:
1. Visual hierarchy and information architecture
2. Navigation clarity
3. Call-to-action prominence
4. Mobile responsiveness indicators
5. Potential accessibility concerns

Provide 5 specific, prioritized recommendations for improvement."
```

### Accessibility Check
```
"Review this page for WCAG 2.1 accessibility compliance. Check:
- Color contrast ratios
- Text readability
- Button and link sizing
- Form labels
- Alternative text for images
- Keyboard navigation

List all issues found with specific fixes."
```

## ðŸŽ¯ Quick Testing Workflow

### Test Any Website in 15 Minutes

**1. Initial Impression (2 minutes)**
- Load the site on desktop and mobile
- Note first impressions: professional? trustworthy? clear purpose?
- Identify the primary call-to-action

**2. Speed Test (3 minutes)**
- Run Google PageSpeed Insights
- Note Core Web Vitals scores
- Identify top issues

**3. SEO Check (5 minutes)**
- View page source (Ctrl+U)
- Check title tag and meta description
- Verify H1 tag
- Test on mobile (Chrome DevTools: Ctrl+Shift+M)

**4. Usability Review (5 minutes)**
- Try to complete a key task (find info, make purchase)
- Rate usability heuristics (1-4)
- Note any frustrations or confusion

**5. AI Analysis (Bonus)**
- Screenshot the homepage
- Upload to ChatGPT/Claude
- Get instant feedback and suggestions
