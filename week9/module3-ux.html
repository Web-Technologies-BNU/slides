Core UX Principles

1. User-Centered Design

Design for actual users, not assumptions. Research, test, iterate.

2. Consistency

Predictable patterns reduce cognitive load and build trust.

3. ACCESSIBILITY

Design for everyone, including users with disabilities. WCAG 2.1 AA minimum.

4. FEEDBACK

Every action needs clear, immediate response. Users should never wonder "did that work?"

5. SIMPLICITY

Remove unnecessary elements. Every feature is a tax on users' attention.

6. ERROR.PREVENTION

Design to prevent mistakes rather than just handling them.
